```
├── src                                   # 源码
│   ├── images                            # 项目资源
│   │   └── ...
│   ├── media                             # 项目资源
│   │   └── ...
```

## 快速开始

* 安装依赖

```
npm install

```

* 起步

```
npm run dev

```


* 发布

```
npm run build

```
